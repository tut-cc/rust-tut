# 基本の文法

コンパイルが通らないプログラムを直していくことで、Rustの文法を少しずつ直していきましょう。

以下のコマンドを実行するとエクササイズを開始できます。

```sh
cd l1_basics
cargo xtask # TODO: まだ実装途中
```

エクササイズには二桁の数字が割り振られています。
トピックごとに二桁目の数字が変わります。

- 00: ハローワールド
- 10~: 変数の宣言と代入
- 20~: 関数
- 30~: コントロールフロー
- 40~: 基本データ型とベクタ型
- (準備中) 50~: ライフタイム
- (準備中) 60~: 列挙型と構造型
- (準備中) 70~: オプション型と結果型
- (準備中) 80~: ジェネリクスとトレイト

90~ では実践編として `nl` というCLIコマンドを作ります。
